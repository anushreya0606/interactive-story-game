# Interactive-Story-Game
It is a game where the user has an interface to interact with and is a joyful and fun game to play with
It is designed to bridge the gap between classic command-line storytelling and modern web interactivity. It features a modular architecture that separates story logic from presentation, allowing for complex branching narratives and dynamic state management
Tech Stack
1.Logic Core: C (Handling game state and decision trees)

2.Frontend Engine: JavaScript (Scene management and DOM manipulation)

3.UI/UX: HTML5 & CSS3

4.Environment: Node.js (Package management via package.json)
